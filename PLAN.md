# BeEF on Render — Deployment Plan

## Architecture

```
┌─────────────────────────────────────────────────────────┐
│                    RENDER.COM                            │
│                                                          │
│  ┌──────────────────────────────────────────────────┐   │
│  │  Web Service (Docker)                            │   │
│  │  Language: Docker                                │   │
│  │  Instance: Starter ($7/mo)                       │   │
│  │                                                   │   │
│  │  ┌────────────────────────────────────────────┐  │   │
│  │  │  BeEF Container                            │  │   │
│  │  │                                            │  │   │
│  │  │  Port 3000 ────→ BeEF Admin UI + Hook URL  │  │   │
│  │  │  Port 61985 ───→ WebSocket                 │  │   │
│  │  │  Port 61986 ───→ WebSocket Secure          │  │   │
│  │  │  Port 6789 ────→ BeEF Proxy (internal)     │  │   │
│  │  │                                            │  │   │
│  │  │  /beef/data ───→ Persistent Disk           │  │   │
│  │  └────────────────────────────────────────────┘  │   │
│  └──────────────────────────────────────────────────┘   │
│                                                          │
│  Public URL: https://your-beef.onrender.com              │
│  Hook URL:   https://your-beef.onrender.com/hook.js     │
│  Admin UI:   https://your-beef.onrender.com/ui/panel    │
│                                                          │
└─────────────────────────────────────────────────────────┘
```

## Files to Create

| File | Purpose |
|------|---------|
| `Dockerfile` | Multi-stage build for BeEF on Render |
| `config.yaml` | Pre-configured with non-default creds, 0.0.0.0 binding |
| `docker-compose.yml` | Local testing before deploy |
| `.dockerignore` | Optimized build context |
| `render.yaml` | Infrastructure-as-Code for Render |
| `setup.sh` | One-command local setup script |

## Known Limitations on Render

| Issue | Mitigation |
|-------|-----------|
| Only one public port exposed | BeEF UI/hook on port 3000 |
| Free tier sleeps after 15 min | Use Starter plan ($7/mo) |
| No raw TCP/UDP | WebSocket works over HTTPS |
| Persistent disk required | 1GB disk at `/beef/data` |
| Build timeout 20 min | Multi-stage build ~5-8 min |
| Hook URL must use HTTPS | Render provides free SSL |

## Deployment Steps

1. Create repo with all files
2. Push to GitHub
3. Connect repo to Render
4. Deploy with Docker runtime
5. Verify hook endpoint loads
6. Login to admin panel
7. Configure modules and autorun rules
